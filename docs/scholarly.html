DoG: Document Web Generator

DoG is a modular web system for generating and managing HTML and PDF documents starting from predefined templates. The project focuses on server-side Web technologies, asynchronous client-server communication and open data formats.

Functional requirements

  • Authenticate users and enforce role-based access.
  • Generate documents from predefined templates using JSON, random data or CSV input.
  • Store generated documents and expose HTML and PDF exports.
  • Provide an administration module for users, templates and audit history.
  • Use SQLite for persistence and fetch-based AJAX for browser communication.
